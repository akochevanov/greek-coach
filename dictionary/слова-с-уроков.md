# Слова с уроков

Слова сгруппированы по части речи (в греческом и русском это не всегда совпадает один в один — у пояснений смотри колонку «Русский»).


## Существительные (οὐσιαστικά)

| Ελληνικά         | Русский   |
| ---------------- | --------- |
| δρόμος (дромос)  | дорога    |
| εβδομάδα (эвдомада) | неделя  |
| κούραση (кураси) | усталость |
| μέρα (мера)      | день      |
| τέλος (телос)    | конец     |


## Глаголы (ῥήματα)

| Ελληνικά                    | Русский             |
| --------------------------- | ------------------- |
| ακούω (акуо)                | слышать             |
| ξεκινάω (ксекинао)          | начинаю             |
| βγαίνω (вгэно)              | выхожу              |
| έχω (эхо)                   | иметь               |
| οδηγώ (одииго)              | водить (машину)     |
| πάω / πηγαίνω (пао / пигено) | идти (одно и то же) |
| περνάω (пернао)             | проводить (время)   |
| προσπαθώ (проспафо)         | пробовать, пытаться |
| σημαίνει (симэни)           | значит, означает    |
| συνεχίσουμε (синехизумэ)    | продолжаем          |
| τραγουδώ (трагуно)          | петь                |
| τρώω (троо)                 | есть, кушать        |

## Междометия и восклицания

В этом списке отдельных «чистых» междометий мало: разговорную реакцию **ωραία!** («классно!», «отлично!») см. в таблице **прилагательных** — там же указаны значения.

При необходимости сюда можно позже добавить слова вроде *μπράβο*, *αχ*, *ωχ* и т.п.
