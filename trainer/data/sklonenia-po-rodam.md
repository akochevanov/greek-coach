# Практика: местоимение + глагол

Словосочетания по словарю глаголов. Настоящее время.

В транскрипции **заглавная буква** = ударная гласная.

| Ελληνικά              | По-русски        | Транскрипция |
| --------------------- | ---------------- | ------------ |
