# Простые фразы (слова с уроков + доп. существительные)

Источник лексики: `dictionary/слова-с-уроков.md`, плюс: **άνθρωπος, άνδρας, αγόρι, κορίτσι, βράδυ, γονείς, δουλειά, λεφτά, χρόνος, οδός, γραφείο**.  
Для связки «кто/что есть каким» используются формы **είμαι / είναι** (в уроковом списке их нет, это опорная грамматика).

В транскрипции **заглавная гласная** = ударная.
